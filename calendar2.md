# TF追星日历

这是一个可订阅的 `.ics` 日历项目，用于同步时代峰峻相关活动到 Apple Calendar / Google Calendar。

## 文件

- `tf-calendar.ics`：日历订阅文件
- `index.html`：GitHub Pages 首页

## GitHub Pages 使用方法

1. 新建一个 GitHub 仓库，比如 `tf-fan-calendar`
2. 上传本文件夹里的所有文件
3. 打开仓库 Settings → Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，Folder 选择 `/root`
6. 保存后等待 1-2 分钟

你的订阅地址会是：

```text
https://你的GitHub用户名.github.io/tf-fan-calendar/tf-calendar.ics
```

iPhone 添加：设置 → 日历 → 账户 → 添加账户 → 其他 → 添加已订阅的日历。
